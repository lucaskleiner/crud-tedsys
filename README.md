# crud-laravel
Desafio Tedsys CRUD Laravel 9 Vite, Autenticação Breeze e CSS Tailwind 


## Recursos do projeto

-   Create Tarefas
-   Read Tarefas
-   Update Tarefas
-   Delete Tarefas


## Instalação

```bash

git clone https://github.com/lucaskleiner/crud-tedsys.git
cd crud-tedsys
composer install
npm install & npm run dev
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve

```
